# 📊 Data Visualization and Storytelling

## 📌 Objective
Build an interactive Power BI dashboard to analyze sales data and present meaningful business insights through visual storytelling.

## 🛠️ Tools Used
- Power BI
- CSV Dataset

## 📂 Dataset
- Superstore Sales Dataset

## 📈 Dashboard Highlights
- Sales, Profit & Quantity KPIs
- Sales & Profit Trends
- Category & Regional Analysis
- Segment & Ship Mode Analysis
- Discount Impact on Profit

## 💡 Key Insights
- Total Sales: **2.3M**
- Total Profit: **286K**
- West region achieved the highest sales and profit.
- Technology was the most profitable category.
- High discounts reduced profitability.

## 📁 Files Included
- Power BI Dashboard (.pbix)
- Superstore Dataset (.csv)
- Dashboard PDF
- README.md

## 🎯 Outcome
Developed an interactive Power BI dashboard that provides clear business insights and supports data-driven decision-making.
